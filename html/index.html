¿Qué Sabés? El regreso

Como ustedes habrán notado nuestro primer proyecto quesabes.uy estuvo fuera de servicio un tiempo.  Las razones fueron un ataque al servidor que sostenía quesabes.uy. Esto nos motivó para por un lado corregir los problemas de seguridad, así como hacer una actualización al software  que soporta nuestro sitio: Alaveteli.  Aunque invisible este trabajo no es ” mágico”.
